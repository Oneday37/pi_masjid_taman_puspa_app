# **MASJID TAMAN PUSPA APP**
### **Role:** UI Designer & Programmer

## ðŸ’¬ About App
Masjid Taman Puspa Apps is an individual project built using the **Flutter** framework and **Firebase** as its database. This application is designed to help mosque administrators manage finances and convey information about activities held by the mosque to local residents. In addition, users can also read the Qur'an through the Qur'an menu provided and view prayer schedules anytime, anywhere.
